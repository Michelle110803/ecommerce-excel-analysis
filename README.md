# E-Commerce Sales Analysis (Excel) 

## Project Overview
Cleaned and analysed a messy e-commerce dataset using Microsoft Excel. Built an interactive dashboard to visualise key business insights. 

## Tools Used
- Microsft Excel (Power Query, Pivot Tables, Charts)

## Data Cleaning Steps
- Removed invalid text in Price and Quantity columns
- Fixed inconsistent date formats
- Standardised Category names
- Removed negative values and duplicates
- Relcalculated Total column from scratch

## Dashboard Includes
- Sales by Category (Bar Chart)
- Orders by Status (Pie Chart)
- Monthly Sales Trend (Line Chart)

## Dataset Source
[Messy E-Commerce Sales Dataset - Kaggle](https://www.kaggle.com/datasets/kandeelai22/messy-e-commerce-sales-dataset)
